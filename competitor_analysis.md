# 나라장터 관련 경쟁 솔루션 분석

## 1. 주요 경쟁사 현황

### 1.1 케이비드(K-BID) - 시장 1위
- **URL**: https://www.kbid.co.kr
- **특징**:
  - 국내 입찰정보 제공 1위
  - 예가사정률 예측 특허 보유 (특허 제10-1436086호)
  - GS 인증 1등급 (인증번호 17-0475)
  - 연간 회원 낙찰 4만건
- **서비스**:
  - 입찰/낙찰 정보
  - 적격심사 자동계산
  - 입찰분석 시스템
  - 전자입찰 교육
- **데이터 수집 방식**: 
  - 공공데이터포털 API + 자체 크롤링 병행 추정
  - 실시간 업데이트 제공

### 1.2 인포21C
- **URL**: https://www.info21c.net
- **특징**:
  - 맞춤형 입찰정보 전문
  - 1:1 분석 서비스 제공
- **서비스**:
  - 입찰서류함
  - 내역서 작성
  - 공동도급 관리
- **데이터 수집 방식**:
  - API 기반 (공지에 API 관련 내용 있음)

### 1.3 비드큐(BidQ) 
- **URL**: https://www.bidq.co.kr
- **특징**:
  - 국내 최초 입찰정보 서비스
  - 낙찰연구소 운영 (총 230조원 낙찰 적중)
- **서비스**:
  - 시설/용역/구매 통합
  - 업체정보 제공
  - 1:1 분석

### 1.4 지투비플러스(G2Bplus) - 무료 서비스
- **URL**: https://www.g2bplus.kr
- **특징**:
  - 무료 서비스
  - AI 분석 기능
  - BigData 활용
- **데이터 수집**:
  - 나라장터(data.go.kr) + 25개 자체조달기관 데이터
  - AI/빅데이터 기술 적용

### 1.5 비드폼(BidForm) - 플랫폼형
- **URL**: https://www.bidform.co.kr
- **특징**:
  - 전문가 매칭 플랫폼
  - 낙찰 전문가 연결 서비스

## 2. 데이터 수집 방식 분석

### 2.1 공통 데이터 소스
```
1. 공공데이터포털 API (주력)
   - 조달청_나라장터 입찰공고정보서비스
   - URL: https://apis.data.go.kr/1230000/BidPublicInfoService04
   - 장점: 합법적, 안정적
   - 단점: 실시간성 부족, 일부 정보 제한

2. 직접 크롤링 (보조)
   - 나라장터 직접 접속
   - 세션/로그인 관리 필요
   - 실시간 정보, 상세 데이터 수집
```

### 2.2 하이브리드 방식 채택 이유
- **기본 데이터**: 공공데이터포털 API
- **상세 정보**: 직접 크롤링
- **첨부파일**: 나라장터 다운로드
- **실시간 알림**: 크롤링 + 푸시
